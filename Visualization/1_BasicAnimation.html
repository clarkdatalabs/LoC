Making Maps in D3

We want to make a map using D3, a pretty common task with plenty of examples. To do this, we will need a TopoJSON representation of country boundaries. TopoJSON is a JSON format for representing geographic data. It is an extension of the GeoJASON format which also includes a topology, meaning features can share boundary points and lines, reducing file size and also encoding adjacency.

  • Learn more about the TopoJSON format

I found a suitable TopoJSON after browsing a few examples of D3 world map examples online. This was my first D3 project, so I used the needed a bit more hand holding to get up and running.
