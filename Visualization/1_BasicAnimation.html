Making Maps in D3

We want to make a map using D3, a pretty common task with plenty of examples. To do this, we will need a TopoJSON representation of country boundaries. TopoJSON is a JSON format for representing geographic data. It is an extension of the GeoJSON format which also includes a topology, meaning features can share boundary points and lines, reducing file size and also encoding adjacency.

  • Learn more about the TopoJSON format

I found a suitable TopoJSON after browsing a few examples of D3 world map examples online. This was my first D3 project, so I needed a bit more hand holding to get up and running. This video gives a great explanation of the whole process of mapping, projections, and binding data to a static map for display:

One thing to note: I used a power scale to determine the color gradiant of countries:

	var countryColor = d3.scalePow()
			.exponent(.2)
			.domain([0,maxCount])
			.range([minColor, maxColor])
			.clamp(true);

The maxCount constant is calculated to be the largest number of records for a single country in a single year. Alternatively, we could use quantized color steps using d3.scaleThreshold, which would make differences between countries more obvious (Bostock example).

Adding Animation

Once we've figured out how to use this countryColor function to color countries statically for a single year, we can add animation by doing the following:
  • Define an updateDraw function that will be called every time we want to alter country color, i.e. every time we change the year.
  • Use d3.interval(updateDraw,timeStep) to call this function every [timestep] milliseconds.
  • Increment the year counter: 
    year = (( (year  % startYear) + 1 ) % (endYear - startYear)) + startYear
